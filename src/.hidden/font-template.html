How to Use

HTML 
Class name with Icon name: (prefix : <%= className %>)
===========================================

Eg: <i class="<%= className %>-<%= glyphs[0].name %>"> </i>

Other Useful Classes :
--------------------------

large : .<%= className %>-lg, 
Scale : .<%= className %>-1x,
Size : .<%= className %>-20, [.<%= className %>- Desire Value (max : 75) but don't use scale and size both]
Fullwidth : .<%= className %>-fw
List Item : .<%= className %>-ul, .<%= className %>-li
Icon Border: .<%= className %>-border
Float : .<%= className %>-pull-left, .<%= className %>-pull-right
Rotation : .<%= className %>-rotate-90, .<%= className %>-rotate-180, .<%= className %>-rotate-270
Flip : .<%= className %>-flip-horizontal, .<%= className %>-flip-vertical 
Stack : .<%= className %>-stack, .<%= className %>-stack-1x, .<%= className %>-stack-2x
Push Left/Right/Top/Bottom : .<%= className %>-l20 [.<%= className %>-use l,r,t,b for left,right,top,bottom and a desire value to push (max : 75)]
SCSS 
Use with Sass @include property,
==============================================

There are two ways to use SCSS attributes


    
        

            Individual Value
            Shorthanding Value
        

    
    
        

            
                Eg: 

  .my_class{

   @include <%= className %>(<%= glyphs[0].name %>); **.2

   @include <%= className %>-size(14px);

   @include <%= className %>-scale(3x); //Max 10x

   @include <%= className %>-color(red);

   @include <%= className %>-left(14px);

   @include <%= className %>-right(14px);

   @include <%= className %>-top(14px);

   @include <%= className %>-bottom(14px);

   @include <%= className %>-align(middle);

   @include <%= className %>-rotate(45deg);

   @include <%= className %>-flip(verticle);

   @include <%= className %>-spin(spin);

   @include <%= className %>-transition(transition);

  }
            		
            
                Eg: 

  .my_class{

     @include <%= className %>-font(

        $insert:before, **.1

        $icon:<%= glyphs[0].name %>,

        $size:14px,

        $scale:3x,

        $color:red,

        $left:5px,

        $right:5px,

        $top:5px,

        $bottom:5px,

        $align:middle,

        $rotate:45deg,

        $flip:verticle,

        $spin:spin);

        $transition:all 300ms ease);

  }

NOTE : Don't use Size & Scale both h
**.1 This $insert value only need if you want to add an icon to the pseudo class. (before/after) default is none or without specifying $insert or blank. -- !optional
**.2 By default, you can use the icon name for this value, if you want more specific you can specify $icon, $insert(before/after or blank) and $extend(boolean) but we recommend to use $icon only. -- !optional
